# The Beauty Blend website

This GitHub Pages package includes two deployment options.

## Recommended option

Upload these items to the root of your GitHub repository:

- index.html
- styles.css
- script.js
- assets folder

Keep the assets folder name and image filenames unchanged.

## Single-file option

Use index-self-contained.html when images fail because the assets folder was not uploaded. Rename it to index.html before uploading. All service images, CSS and JavaScript are embedded inside the file.

## GitHub Pages

1. Open repository Settings.
2. Open Pages.
3. Choose Deploy from a branch.
4. Select main and root.
5. Save.
