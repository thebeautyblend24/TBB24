# The Beauty Blend Website

A responsive, accessible website for The Beauty Blend. The site works on GitHub Pages and sends appointment requests to WhatsApp.

## Files

- `index.html`
- `styles.css`
- `script.js`
- `assets/logo.png`
- `assets/hair.jpg`
- `assets/nails.jpg`
- `assets/mehndi.jpg`

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload every file and folder from this project.
3. Open the repository Settings.
4. Select Pages.
5. Under Build and deployment, choose Deploy from a branch.
6. Select the `main` branch and `/root` folder.
7. Save.
8. GitHub will show the public website address after deployment.
